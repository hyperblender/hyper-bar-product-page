Where To Find It

After enabling the add-on, open the 3D Viewport sidebar with N and go to the Hyper Bar tab.

The progress bar is available in two places: inside the 3D Viewport sidebar and, optionally, in the render window header.

  • Main render progress bar.
  • Live metrics during rendering.
  • Recent render history.
  • Render Animation button.
  • Trash icon button for clearing render history.
  • Small preferences shortcut in the panel header.

Starting A Render

You can start rendering in the usual Blender ways. Hyper Bar does not use a separate custom render system.

  • Use Blender's native Render Animation command.
  • Press the Render Animation button in the Hyper Bar panel.

The Render Animation button is only a shortcut for Blender's native animation render. While a render is active, the button is disabled to prevent starting another render by accident.

Main Progress Bar

The main progress bar shows the current render status. For animation renders, it shows render percentage, rendered frames and total frames, and ETA when enough data is available.

At the beginning of an animation render, Hyper Bar shows Calculating ETA.... After the first finished frame, the add-on can start estimating the remaining time.

25%, 30/120, 12:40 left

When the animation is complete, the progress bar stays at 100%.

Live Metrics

Live Metrics appear only while a render is active. These values are most useful for animation rendering.

Elapsed How long the current render has been running.
Avg Frame Average time per frame, based on recent rendered frames.
Finishes At Estimated clock time when the render should finish.
Smooth ETA The ETA uses recent frame times, so it can adapt when frame render time changes during the sequence.

Render History

Hyper Bar keeps a short history of recent render sessions. The history stores up to 10 recent render sessions and can be cleared with the trash icon button next to Render Animation.

  • Session number.
  • Rendered frame range.
  • Finish time.
  • Total elapsed time.
  • Average time per frame.

Completed renders are shown normally. Canceled renders are kept in the history too, but they are displayed in a muted style.

Add-on Settings

Open Blender Preferences, find Hyper Bar in the add-ons list, and open its settings. You can also use the small settings shortcut in the Hyper Bar panel header.

ETA Recent Window

This setting controls how many recent frames are used to calculate the average frame time and ETA.

Default value: 10

A smaller value reacts faster to changing frame times, but the ETA may jump more. A larger value gives a smoother ETA, but it reacts more slowly.

Show In Render Window

This setting controls whether Hyper Bar is also shown in the render window header.

Default value: On

Turn it off if you want to keep the render window header cleaner.

Notes

  • Hyper Bar uses the scene frame range and frame step to calculate the total number of animation frames.
  • The add-on updates the UI automatically during rendering.
  • The add-on is lightweight and keeps render calculations separate from Blender's actual rendering process.
